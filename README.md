Some university tasks for better c++ understanding.
---
**Rational**

Creating rational numbers like a/b, providing main operations: +, -, *, /

**MatrixS**

2-dimensional matrix with basic operations, getting an element and resizing.

**ArrayD**

Growable array of doubles. Supports several operations, including: get an element, resize, insert, remove.

**ArrayT**

The same as ArrayD, but of any type, uses template.
